在ubuntu上eclipse连接postgresql

发表于 2016-07-05 | 分类于 安装 |

ubuntu安装postgre

环境

  • ubuntu14.04.4
  • eclipse-cpp-juno-SR2-linux-gtk-x86_64
  • jdk-8u91
  • postgresql-9.2.2
  • g++

安装ubuntu

  • 复制粘贴拖曳
    • Virtualbox-Devices-Shared Clipboard-Bidirectional
    • Virtualbox-Devices-Drag and Drop-Bidirectional
  • 全屏显示
    • Virtualbox-Devices-Insert Guest Additions CD image（需重启）

安装eclipse

安装java

  • sudo add-apt-repository ppa:webupd8team/java
  • sudo apt-get update
  • sudo apt-get install oracle-java8-installer
  • sudo apt-get install oracle-java8-set-default

安装PostgreSQL

  • biliyuan@biliyuan-VirtualBox:~/project/postgresql-9.2.2$ ./configure –prefix=/home/biliyuan/project –enable-depend –enable-cassert –enable-debug

安装g++

  • 处理problem：g++ not found in path
    • Installing g++ The GNU C++ complier using ubuntu software centre：sudo apt-get install build-essential
    • Changing in Window -> Preferences -> C/C++ -> Build -> Settings -> Discovery -> CDT GCC Build in Complier Settings [Shared]
      • From: ${COMMAND} -E -P -v -dD “${INPUTS}”
      • To: /usr/bin/${COMMAND} -E -P -v -dD “${INPUTS}”
